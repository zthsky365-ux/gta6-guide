# GTA6 战斗技巧完全指南

## 掩体系统

GTA6 的掩体系统比前作更加智能：

1. **自动掩体** - 靠近障碍物会自动寻找掩体
2. **手动控制** - 使用 Q 键手动进出掩体
3. **快速移动** - 在掩体间快速切换（Space + 方向键）

## 射击技巧

### 瞄准辅助

- **控制器玩家** - 开启辅助瞄准
- **键鼠玩家** - 练习预瞄和跟枪
- **难度调整** - 在设置中可以调整辅助强度

### 武器选择建议

| 场景 | 推荐武器 | 理由 |
|------|-----------|------|
| 近距离 | 散弹枪 / SMG | 高爆发伤害 |
| 中距离 | 突击步枪 | 稳定输出 |
| 远距离 | 狙击枪 | 一击必杀 |
| 载具战 | 火箭筒 / 重机枪 | 对载具有高伤害 |

## 进阶技巧

### 快速切换武器

设置武器轮盘快捷键，战斗中快速切换适合当前场景的武器。

### 手雷使用

- 预判敌人位置投掷
- 使用按住键（默认 G）调整投掷距离
- 粘性炸弹可以贴在载具上

### 近战技巧

- 潜行模式下可以一击必杀
- 使用环境物体作为武器
- 拳击适合不会引起警报的场景

## 在线模式 PVP 建议

1. **熟悉地图** - 知道掩体和捷径位置
2. **团队配合** - 不要单独行动
3. **经济管理** - 死亡会掉落武器和金钱
